zkap-circuit WASM Example

This page exercises the three functions exported from bindings/wasm/src/lib.rs:

  • generatePoseidonHash(inputs) — hash an array of field elements
  • generateAudHash(audiences) — hash an audience list (used in the ZK circuit)
  • generateAnchor(secrets) — derive an anchor from identity secrets

Setup: run wasm-pack build --target web inside bindings/wasm/, then serve this directory with any static HTTP server (e.g. python3 -m http.server 8080) and open http://localhost:8080.
